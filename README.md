# Electronics Goods Repairment Shop Management System 🛠️📺

Bu proje, elektronik eşya tamir atölyelerinin iş akışlarını dijitalleştirmek, müşteri ve cihaz takibini optimize etmek amacıyla Java (NetBeans) ortamında geliştirilmiş bir yönetim sistemidir.

---

### 🇹🇷 Proje Özellikleri (Turkish)
* **Kapsamlı Yönetim:** Müşteri kayıtları, arızalı cihaz takibi ve onarım durum güncellemelerini içeren uçtan uca bir yönetim paneli.
* **Veritabanı Mimarisi:** Veriler, **MySQL** üzerinde saklanmakta olup tablolar arası ilişkiler **Primary Key (PK)** ve **Foreign Key (FK)** yapılarıyla kurgulanmıştır.
* **İş Akış Otomasyonu:** Tamir aşamasındaki ürünlerin (Beklemede, Tamirde, Teslim Edildi) durum takibi kolaylıkla yapılabilmektedir.
* **Kullanıcı Arayüzü:** Java Swing kullanılarak tasarlanmış, kullanıcı dostu ve fonksiyonel bir GUI (Arayüz).

### 🇬🇧 Project Features (English)
* **Comprehensive Management:** An end-to-end management dashboard for customer records, faulty device tracking, and repair status updates.
* **Database Architecture:** Powered by **MySQL**, with a relational schema utilizing **Primary Key (PK)** and **Foreign Key (FK)** constraints for data integrity.
* **Workflow Automation:** Easily track the status of products (Pending, In Repair, Delivered) throughout the service cycle.
* **User Interface:** A user-friendly and functional GUI designed using Java Swing.

---

### 🛠️ Teknolojiler / Technical Stack
* **Language:** Java (JDK)
* **IDE:** NetBeans
* **Database:** MySQL (Relational Design)
* **GUI:** Java Swing / AWT
